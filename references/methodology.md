# 柴西 Vlog 拍摄方法论（规划器注入用）

> 提炼自博主「超Carry的柴西」的 Vlog 制作教学视频（含其逐字讲解）。本文件是 `vlog-planner` 的"大脑"——生成逐日拍摄手册时，规划器会把这里的原则注入到每个景点的建议里。本文为方法论复述与转化，非原视频转录。

视频骨架（8 章）：内容策划 · 外景 APP 推荐 · 行程规划 · 拍摄设备 · 三镜头拍摄法 · 剪辑软件 · 后期流程 · 心得。本 skill 聚焦前三章 + 三镜头拍摄法。

---

## 1. 前期内容策划：两个阶段

**阶段一 · 提供内容（客观事实）**
收集整理偏科普/公设的基础信息：怎么去、多少钱、**景点的渊源与由来**、路线怎么安排。她的提醒：

- 公设攻略在网上已相当饱和，地球上的热门地方几乎都能找到攻略——所以**客观信息不是壁垒，要靠它打底但别止于此**。
- 对"众说纷纭"的说法**多问 AI 考证**，别照搬。
- 团队用**飞书**多端协作整理信息（编导先整理，她再从中找视角发散）。

**阶段二 · 找到自己的风格与视角（主观）**
创作者之所以是创作者，是因为带着**主观视角**去拍。她的核心洞察：

> **再美的风景，如果没有故事发生，也很难被人记住。** 她最珍贵的素材，往往是旅行中**和人发生的故事**。

所以策划时要想：**什么能真正触动你、引发你的感受**。一条视频不可能满足所有观众，不必讨好所有人——选定自己的视角，吸引同类人即可。

**过程即故事**：A-roll 是叙事主线，别只拍高潮结果——**早起、赶路、爬升、找机位、守候**这些"铺垫/苦"本身就是最抓人的张力（"为热爱早起"的反差、"到底出不出片"的未知）。写手册时，给这些**过程节点也配 A-roll**，不要只给终点。

> 规划器用法：为每个点生成 1–2 个 **A-roll 故事钩子** = 一句"客观事实/由来" + 一个"情绪落点/可能发生的故事"；并给**抵达/爬升/守候等过程节点**各留一个 A-roll 口播点。

## 2. 选景与光线：用工具预判天气、光照、日出日落（甚至潮汐）

"每天出去一天都是钱，所以合理安排行程和**拍摄窗口**非常重要。" 想拍到好画面，必须提前考证 **天气、光照、潮汐、日出日落**。她推荐的工具 → 本 skill 数据源：

她明确列出的四个考证要素是 **天气、光照、潮汐、日出日落**。她推荐的工具 → 本 skill 数据源：

| 她用的工具 | 作用 | 本 skill 对应 |
|---|---|---|
| **Windy** | 按定位精确看云图/降雨/风，几小时~几天比别的软件更准；部分地区还能调当地**摄像头**看实况 | 天气 provider（默认 Open-Meteo）；实况见下方"实况验证" |
| **Skyline**（网站） | 不出门看世界各地**实时风景**（如"今天意大利草绿不绿") | POI/选景 provider（默认 OSM）；实况见下方"实况验证" |
| **Lumy** | 精准定位日出日落——**拍人像最佳时间是日落前 1 小时（Golden Hour）** | 内置太阳计算 |
| **Lumos** | **每天每小时太阳在哪个位置** → 安排机位与拍摄内容 | 内置太阳计算（含方位角） |
| （**潮汐**） | 她把潮汐与天气/光照并列考证 | `marine.py`（海况：浪高/涌浪/海温）+ 潮汐表另查 |

关键判断：**"好看日落很吃天气"**——这正是规划器"出片指数"要量化的：黄金时刻 + 低降水概率 + 适度云量 = 高分。

**潮汐 / 海况（海边拍摄必看）**：海边/赶海/滩涂/海岛日出要看潮汐——**低潮**拍滩涂倒影、赶海、礁石；**涨潮**拍浪。`marine.py` 给浪高/涌浪/海温（安全 + 浪的戏剧性），但**潮汐涨落时刻要另查当地潮汐表**（`WebSearch "<地点> 潮汐表 <日期>"`）。内陆点跳过。

**实况验证（临行前一步，别只信预报）**：柴西用 Windy 摄像头、Skyline 看**实时**画面。本 skill 的天气/海况都是**预报**——**出发前 1–2 天，用 `WebSearch` 查目的地的"景区直播 / webcam / 实况"核对**（很多日出/雪山景区有直播），预报与实况打架时以实况为准、并据此微调。

## 3. 行程规划：把每天定性为 A-roll日 / B-roll日 / 赶路日

她的长行程（如 22 天自驾贝加尔湖）做法——**提前把每天定性**：

- **A-roll 拍摄日**：天气光线好、有故事点的重点日。**A-roll = 叙事主线，决定视频的"下限"**：故事好，哪怕手机随手拍也精彩。
- **B-roll 拍摄日**：补环境/空镜。**B-roll 拍得好，提升"质感"与"上限"**。
- **坏天气日 / 垃圾时间**：直接**赶路、转场、休整**，不硬拍。

"最佳路线"是**本人 + 摄影 + 制片 + 编导四方辩论**出来的最优解（最具性价比）——争论点常是"再多两天拍空镜""这天要拍日落但天气不稳"。

**短行程同理**：哪怕只有一天，也按**时段**安排——"早上五点起来先拍日出，九点景点开门再拍……"。

> 规划器用法：依据天气/光线分数给每天打一个**角色建议**（A-roll重点日 / B-roll空镜日 / 赶路休整日），并把"坏天气日"的时间退给赶路或室内/情绪向拍法。

## 4. 三镜头拍摄法（到一个想记录的地方，快速拍齐一组）

> ⚠️ 这是柴西的具体方法，**不是**泛泛的"远景中景特写"。目的：短时间得到一组**相对完整**的镜头，最大程度保留当下信息，方便后期串故事、不遗漏。

1. **镜头一 · 自拍**：来一个你自己的画面，**记录客观事实**或**抒发情感**。
2. **镜头二 · 你的眼睛（主观视角）**：把镜头当成你的眼睛——你**看到了什么**、想记录什么、**什么让你有感受**；用主观视角增强代入感。
3. **镜头三 · 把你放进环境里**：千万别忘记拍一个**人在环境中**的画面（人 + 场景）。

> 规划器用法：每个点都按这三步各给 1–2 个具体镜头建议。

## 5. 设备（按需，配置可改）

主力 **Sony FX3 + ZV-E1**（同 CMOS，便于统一调色）；镜头大三元 + 秘密武器 **FE 2/28-70 GM（超三元，大光圈变焦，旅拍轻便）**；无人机 **Mavic 4 Pro**；**Osmo Pocket** 做辅助镜头；**Sony DV** 拍复古"活人感"。个人轻装可只带手机 + 一机一镜。她的原则：**长期主义**（一步到位买对，别小手小脚走弯路）。

**装备 → 场景映射**（写手册时据此给每个点挑设备，而不是笼统列一堆）：

| 想要的画面 | 用什么 |
|---|---|
| 定场 / 星空 / 狭窄空间 / 广阔大地 | **广角** |
| 雪山压缩 / 日轮特写 / 远处主体 / 人物半身 | **长焦**（大光圈更佳） |
| 公路 / 云海 / 大地纹理 / 高位环境 | **无人机**（大风/雨/雪季机会少，注意禁飞区） |
| 徒步跟拍 / 运动 / 手持稳 | **Osmo Pocket** |
| 复古"活人感"人物 | **DV / 胶片感** |
| 一镜走天下（轻装旅拍） | **一机 + 大光圈变焦（如 28-70）** |
| 延时 / 日出日落色彩渐变 | **三脚架 + 相机/手机延时** |

## 6. 后期与心得（手册可附带提醒）

- 流程：**Final Cut Pro 剪辑 → 达芬奇调色**；个人/笔记本小屏推荐 FCP（把精力花在**故事和创意**上，而不是对抗软件卡顿）。
- 素材：整理 → 粗剪（"切菜"）→ 文案/配音/音乐（"炒菜"）。**声音设计**对成片影响很大。
- 心得：**所有大师都从模仿开始**；取法乎上；"想全是问题，做才有答案"——先拍先剪，遇到问题一个个解决。
